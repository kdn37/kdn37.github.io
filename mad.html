Mobile Application Developer

Thirty Seven
Work from Home
Full Time

The Mobile App Developer will work with a team on an application centred around suggesting recipes and creating shopping lists for meals. The app will have a data base of current ingredients the user, recipes, and stock of local supermarkets. The app will base the recipe suggestions and creating shopping lists using the databases of current ingredients the user has as well as preferences.

UI/UX Design
Application Security
Database management
Source control management tools e.g., GIT, SVN
